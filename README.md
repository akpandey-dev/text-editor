# Text Editor

A versatile, rich-text editor designed for quick note-taking and text formatting. This is a WYSIWYG editor that is still evolving.
Save your work instantly as HTML with one click.

⚠️ Note: This is a client-side editor intended for personal, educational, and experimental use. It does not aim to replace full-scale word processors.

## Features:

- Bold
- Italics
- Code blocks
- Save as HTML
- Ctrl + S save option
- Clear Formatting
- Clear Editor
